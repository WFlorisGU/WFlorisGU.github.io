下面记述一下，如何在vba中实现汇总word文章中不同表的内容资料：

` Sub 汇总所有数据() '定义所有变量 Dim ff As String, path As String, a As Integer, Filename As String '定义等会窗口中过滤出的文件类型 ff = "Word文档(*.doc;*.docx),*.doc;*.docx " '窗口中左上角显示文字 strtitle = "打开Word文档" '选择后取路径，并赋值到path中 path = Application.GetOpenFilename(filefilter:=ff, Title:=strtitle) '将“路径”中的路径和文件名整体分开 a = Len(path) For i = 1 To a '从右开始直到截取到\，才执行if程序中内容 If Mid(Right(path, i), 1, 1) = "\" Then path = Mid(path, 1, a - i + 1) Exit For End If Next '按照dir排序path所有文件名 Filename = Dir(path) '循环查找word文件中table中的指定值，输出到excel中 k = 2 '开始循环，文件不为空，直接开始循环 Do While Filename <> "" '下面这一条固定，表示word应用程序（docapp）被创建,同样每一循环都必须打开word Set docapp = CreateObject("Word.application") '创建对象，打开word文档 Set dd = docapp.documents.Open(path & Filename) '创建对象，附上文档中的第一个表格table Set ddd1 = dd.Tables(1) Set ddd2 = dd.Tables(2) '计算表格中行数 x = ddd2.Range.Rows.Count For i = 2 To x 'i为word中表格的行数 Cells(k, 1) = Mid(ddd1.cell(1, 2).Range, 1, Len(ddd1.cell(1, 2).Range) - 2) Cells(k, 2) = Mid(ddd2.cell(i, 2).Range, 1, Len(ddd2.cell(i, 2).Range) - 2) Cells(k, 3) = Mid(ddd1.cell(1, 4).Range, 1, Len(ddd1.cell(1, 4).Range) - 2) Cells(k, 4) = Mid(ddd1.cell(1, 6).Range, 1, Len(ddd1.cell(1, 6).Range) - 2) Cells(k, 5) = Mid(ddd1.cell(2, 4).Range, 1, Len(ddd1.cell(2, 4).Range) - 2) Cells(k, 6) = Mid(ddd1.cell(3, 4).Range, 1, Len(ddd1.cell(3, 4).Range) - 2) Cells(k, 7) = Mid(ddd2.cell(i, 3).Range, 1, Len(ddd2.cell(i, 3).Range) - 2) k = k + 1 'k为输出到的表格行数 Next '开完一个文档后，各种关闭和归零 dd.Close docapp.Quit Set docapp = Nothing Set dd = Nothing Set ddd1 = Nothing Set ddd2 = Nothing '文件到下一个 Filename = Dir Loop '所有文件读取完后，手动修改第一行 Cells(1, 1) = "姓名" Cells(1, 2) = "日期" Cells(1, 3) = "性别" Cells(1, 4) = "学历" Cells(1, 5) = "工资" Cells(1, 6) = "年龄" Cells(1, 7) = "销售客单" End Sub
