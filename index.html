下面记述一下，如何在vba中实现汇总word文章中不同表的内容资料：

` Sub 汇总所有数据()
‘定义所有变量
Dim ff As String, path As String, a As Integer, Filename As String
‘定义等会窗口中过滤出的文件类型
ff = “Word文档(.doc;.docx),.doc;.docx “
‘窗口中左上角显示文字
strtitle = “打开Word文档”
‘选择后取路径，并赋值到path中
path = Application.GetOpenFilename(filefilter:=ff, Title:=strtitle)

‘将“路径”中的路径和文件名整体分开
a = Len(path)
For i = 1 To a
‘从右开始直到截取到\，才执行if程序中内容
If Mid(Right(path, i), 1, 1) = “\” Then
path = Mid(path, 1, a - i + 1)
Exit For
End If
Next
‘按照dir排序path所有文件名
Filename = Dir(path)

'循环查找word文件中table中的指定值，输出到excel中
k = 2
'开始循环，文件不为空，直接开始循环
Do While Filename <> ""
 '下面这一条固定，表示word应用程序（docapp）被创建,同样每一循环都必须打开word
 Set docapp = CreateObject("Word.application")
 '创建对象，打开word文档
 Set dd = docapp.documents.Open(path & Filename)
 '创建对象，附上文档中的第一个表格table
 Set ddd1 = dd.Tables(1)
 Set ddd2 = dd.Tables(2)
 '计算表格中行数
 x = ddd2.Range.Rows.Count
 For i = 2 To x    'i为word中表格的行数
  Cells(k, 1) = Mid(ddd1.cell(1, 2).Range, 1, Len(ddd1.cell(1, 2).Range) - 2)
  Cells(k, 2) = Mid(ddd2.cell(i, 2).Range, 1, Len(ddd2.cell(i, 2).Range) - 2)
  Cells(k, 3) = Mid(ddd1.cell(1, 4).Range, 1, Len(ddd1.cell(1, 4).Range) - 2)
  Cells(k, 4) = Mid(ddd1.cell(1, 6).Range, 1, Len(ddd1.cell(1, 6).Range) - 2)
  Cells(k, 5) = Mid(ddd1.cell(2, 4).Range, 1, Len(ddd1.cell(2, 4).Range) - 2)
  Cells(k, 6) = Mid(ddd1.cell(3, 4).Range, 1, Len(ddd1.cell(3, 4).Range) - 2)
  Cells(k, 7) = Mid(ddd2.cell(i, 3).Range, 1, Len(ddd2.cell(i, 3).Range) - 2)

  k = k + 1      'k为输出到的表格行数
 Next
 '开完一个文档后，各种关闭和归零
 dd.Close
docapp.Quit
 Set docapp = Nothing
 Set dd = Nothing
 Set ddd1 = Nothing
 Set ddd2 = Nothing
 '文件到下一个
 Filename = Dir
 Loop

 '所有文件读取完后，手动修改第一行
 Cells(1, 1) = "姓名"
 Cells(1, 2) = "日期"
 Cells(1, 3) = "性别"
 Cells(1, 4) = "学历"
 Cells(1, 5) = "工资"
 Cells(1, 6) = "年龄"
 Cells(1, 7) = "销售客单"

End Sub `

首次演讲比赛有感

发表于 2017-12-02

说来也是神奇。去年的心愿，由今年的我来完成。一种很“命运”的感觉。
也是幸亏了我的每月总结，感觉自己去推子自已狠狠地一把。一鼓作气报了这个演讲比赛。有时候就是不需要给自己太多时间去思考，想做就去做。别给自己找太多理由。

感觉自己在这次的演讲比赛中收获的不只是与演讲方面的东西。还学到了一些关于思维格局方面的东西。下面就主要记叙自己印象深刻的点吧。

演讲方面：

总结而言：走心内容+适量技巧。
走心内容：
主要是你自己在演讲的时候，你自己都会被你自己打动。真正的touch到你自己。去演讲一些真正走心的东西。这个过程就是去深度的去挖掘到你自己，自己感受最强的东西，就是我们可以去表达的东西。再者，一些所谓的手势，节奏自然而然就会自动出来了。
适量的技巧：
1.气场
这一点是我感受最大的（某一方面的原因也是Grace老师以我为代表说的）。一直明白，自己这方面的成长空间特别的大。也总是被建议台风可以更稳一些的，但每次的应激反应又是那种hold不住全场了。气息更加稳。
多去看那些个小的女星hold住全场的视频，如孙燕姿的演唱会视频，去带到那种感觉中。多去利用带练平台，去练习。把每一次的平台都好好利用珍惜它，表现你自己。
2.节奏
演讲的节奏，快慢都是两面的，更多的去看你想要表达什么。去利用好你自己的稿子，这是一个很好让大家进入到你的感觉的方式。
当然，可以多去看看优秀的美国人演讲视频。去感受其中的节奏感。
3.手势
这个同上条，将节奏和手势结合在一起。
多去感受和模仿演讲视频里面的手势节奏。当你适应了在演讲中使用手势，你就不会那么别扭，更加自然了。
4.互动
适当的互动是很好的！看每个人的稿子准备了。
5.前期的准备

  1. 稿子
    此处建议去看看外研社英语演讲比赛，优秀选手的稿件，去看看他人的稿件里面的逻辑性，那种结构。
  2. 句型的应用
    这个是真的让你的演讲比分享更加正式的关键点。同时是一种将你所学的东西真正用出来的过程。把东西化成你自己的。
思维方面：

这个方面真的收获了太多的能量。感受与思维格局的打开真的很相关。

第一，全感。
感觉的打开，对你很多方面都是非常有益处。你所喜欢擅长的东西，往往是你感官打开最多的方面。相应的你所不擅长，甚至是讨厌的点，就是你可以去打开更多程度的感官。
从李通的身上，就可以看出他对模仿这个维度的打开，他能感受大英语过程中不一样的东西，所以他的英文成长会更快。这也是Grace老师，说的全感练习的点：就是全感练习，不是一味的机械性学习，否则你会慢慢失去了英文学习的乐趣。去多维度的学习，如“听”——进入到音频感觉中，“模仿”——到你那个模仿的视频感觉里，“唱”——那种节奏那种心情里面等等等。
你敞开越多，感受越多，你身上很多不同维度的东西就会被打开。突然发现人真的是一种很有“潜力”的生物。

第二，看待事物的两个视角。
语速的快和慢，好与坏。就是很典型的两个方面，不同人的角度看待就是不同，可能认为好厉害，也可能认为失去了一种节奏感，只是一味地快。想到Audrey老师看到电影《正义联盟》两个英雄看待自己super power为“accidence”，认为自己像个“monster”，而我们认为他们就是拥有很棒能力的英雄。这就是一件事情看法不同，结果不同。
一件事情，就在那里，只是看你能不能好好看待它，利用好它。

第三，深掘一些东西背后更本质的东西。
你“喜欢音乐”，你喜欢的可能是背后的“轻松”，你觉得别人好棒，然后“模仿”别人身上一些很表层的东西，没有看到TA身上是不是在某一方面特别“打开”。去深掘一件事情里面更核心的东西，不是只停留在表面。做一个“有心人”。

其实还有好多想要记述下来的，例如自己的不紧张，Higher me，令我印象深刻的演讲等等很多很多。但是我上面写的真的就是我想要记下来的，想要去不断成长，挖掘自己的点。 加油加油
踏踏实实
知行统一

十一月份— 起起伏伏

发表于 2017-11-30

昨天还在自己的日记本上写下了“很快很快 十一月也要走了呢 一年时间真的好快啊”
就到了总结十一月份的时候了。

看着图片的我突然觉得自己也没有干什么事的感觉。这样是自己又没有目标了吗？不知道，真的需要去好好拷问自己。自己最近真是越来越不懂自己了。

考试：
这个月真的就是考试周啊。各种考试，自己有感觉考的好的也有坏的。最重要的知道了：重复的重要性，自己要去不断的复习概念，练习例题。反复去确认成长。大学的很多考试，都是这样，你需要把老师上课讲的内容都好好消化，定期总结，这样最后复习也会轻松些。其次就是关于考试提前交卷问题。这个也是要去反复确认几遍，再上交，不然自己之后会后悔的（T0T不要问我怎么知道的……）。关于考试策略，自己本来不想去写这个的，但是，基本的道理我自己也没有把握好……“先做重要的题，就是所谓比分大的题”（来自系统工程考试）
自己现在心态真是平和了好多。也知道自己的成绩都是自己选择的原因，没有什么好埋怨的。很多东西都是有自己的心态决定的。
“尽人事，听天命”

英语：
其实这里更应该分为社团和布莱恩两个方面吧。
自己学着学着总觉得没有那么纯粹了，很多事情的决定都开始由外界决定。很分不清自己的真实的想法。哪个到底是我自己想要的呢。
++ 社团： ++回到了社团。都亏了自己的“不要脸”吧，很快也是一点点回到了这里。还是一种好幸福的感觉。更单纯一点吧。在黑夜中操场上自己一遍一遍背诵自己背过的文章，那一刻自己就知道了英语对于我自己的意义。我真的可以更勇敢一些的。总是被自己绊住步伐。害怕就不敢前进。
++ 布莱恩： ++发音班：让我又找到了那种最初的感觉，就是那种踏实前进，踏实练习的感觉。不要给自己太多的接口了。自己给自己的理由真的是太多的。在发音班里面自己也知道自己可以通过自己的行动，工作给他人带来帮助。这种感觉真的太好了。实战班：实战班去到长春。一个新的开始，打开了我对英语另一个维度的理解吧。英语真的是可以随时随地用出来的，当然只要你想！长春这次其实还是有遗憾的，自己还是太藏着压着，很多东西没有去把握着。给自己一些“强迫”去突破自己，这样自己就会有了成长的感觉。回来了，感觉真好。

其他：
这一部分更多是考研和心理学吧。有些时候自己总觉的有些话真的不要讲太早，就是怕，我自己也不是很懂为什么。计划不要定太早，话不要讲太大。先做事，比什么都重要。加油吧，不要被困在里面。
去蹭了心理课嘻嘻嘻嘻

自己这一个月事情没有做太多，看事情，做事的状态也是在慢慢调整，唯一缺憾的好像是自己越来越看不懂自己了。真的就是这样的……大概记述下自己，心态上面的调整吧。
意识打开的重要性 生活是做减法，不是加法。减法让自己更加舒服。各种生活状态就是这样。
*主动成长，不要给自己太多借口。

想不起来啦。就这样吧，越来越好。
至少要和自己做好朋友。

10月份总结

2017年11月1日 星期三
7:17

本来想先写个大纲的结果……
1.个人Blog
2.十一和爸妈一起出游
3.比赛项目
4.社团
5.网站

昨天思考了一下，差不多这一个月里，做了这些事情，总是会觉得自己的时间不够用。这是我值得反思的一件事情。大概来总结一下吧。
每一件事情都给了我或多或少的启示吧。

个人Blog：从大二开始一直就在想的东西。终于建好了，真的是满满的成就感。只是建好了不能不用。怎么把博客的运用弄好，这也是很大的问题啊。下一步：逐步开放各个页面的功能，每次学以后直接用编辑软件做笔记。

十一和爸妈出游：很开心他们能够过来，突然意识到，自己还是不能够离家真的太远，偶尔的一次暑假不回家，表面上，看上去云淡风轻，好像没有什么事情发生，但是自己心里面遇到的压力，好像并没有小到哪里去……这次旅行，让我感受到了旅行的好处：多去发现不一样的自己，去体验各种突发的状况。和家人的关系更加的打开，更加能够理解他们。下一步：多自己出去玩，出游。

比赛项目：被奖学金刺激的我，报了个比赛。发现只要你主动去做事，去承担东西，收获总是会来到你身边。自己是一个很怕承受责任的人，有时候更多的不想去承担责任，开始各种逃避现实。逼着自己成长的感觉其实还是很不错的。比赛项目学到了很多企业知识还有就是电子商务的各种东西操作，各种营销手段什么的。当然这次更加意识到一个团队的重要性，队长和队员，职责的明确分工。团队的协作能力。你能不能服众，这么去协调团队里面人的关系还有压力。这些都是可以锻炼到的。这些东西在制作网站的过程中也是被我意识到。很重要啊。

网站：既然提到了网站那么就来写一下网站。虽然是网页设计这门课的作业。但是我觉得我做的是相当的仔细。很多时候，都是有种我在为自己而做的感觉。更多的能够感受到自己敲代码的那种满足和开心的感觉吧。做出来一个完整的页面真的是超级的开心。当然一路上磕磕绊绊总是会遇到不少的问题。像是自己每一次的制作都是边想边设计的，都没有很大体的提前想好，没有从宏观到微观，值得反思，这个是很需要去学习的一个点。其次就是上面提到的团队协作的问题。这次有些像是我是队长，然后我是主要负责这些东西的制作的，但是我好像没有针对每个人去给出任务，导致有些人的不耐烦了。下次应该征求他人的意见，他们大概能够把握哪模块的制作，再去给任务，同时给出截至时间。这就是我想要记叙的吧。

社团：有种因小失大的感受。然后值得我反思的就是我对英语的喜爱，是因为社团还是什么，为什么现在放弃了那么多。想起《尼采治焦虑》里面

也不知道是不是自己啰啰嗦嗦敲了一大堆没有用的，总结思考的模式需要更加的清晰，不过呢，我很喜欢这样反思的自己，总之加油啦。
