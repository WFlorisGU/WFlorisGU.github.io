html标记语言

HTML是用于创建网页的语言。
HTML是一种超文本标记语言.通过使用HTML标记标签创建html文档来创建网页。

网页组成

  • DOCTYPE element
    主要告诉浏览器它是个html文件。
  • html element
    • 结构
    • 标记和元素
      标记一般成对存在，但也存在单个标记。如 ‘
      ’分段标记。
      对于双标记，一般形式为：
      <标签名>xxxxx</标签名>
      前者叫开始标签，后者叫结束标签。
      对于单标记：一般会存在“空元素”，即无内容。
    • 属性
      如果把标记和元素比作一个人的骨头，那么属性就像是血肉。它让整个页面更加好看丰满。
      多个属性间，用空格隔开。
      属性是否作用于整个页面：全局属性，局部属性。
      属性是否存在值：一般属性，布尔属性。 布尔属性一般用于属性功能禁用的时候，
    • 其他
      父元素与子元素。
      一个父可以有很多子。子元素建互为兄弟元素。
      标记名不区分大小写。不区分双引号和单引号。
      HTML的实体：不可用平时符号表示的元素。
