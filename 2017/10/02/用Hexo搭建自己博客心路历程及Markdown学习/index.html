Hexo搭建博客及Markdown学习

October 2, 2017 本文总阅读量次

在敲完第一篇正式发上来的文章后，自己思考了下，应该大概记录下自己昨天一天的搭建路程以及Markdown的一些学习记录吧

Hexo搭建之路小述

学校放国庆，自己的时间还是很多的，看完电影后，便决定开始我的Hexo学习之路啦。以下主要记述本人的感受：

首先，很感谢网上的各种资源攻略的。自己最后明白：还是应该先丰富收集资料，再去做事情。所谓“工欲善其事必先利其器”。这一点是我血淋林的教训。自己一共建了两次才建成功并且托管于github。

其次就是接触到了更多的东西。无论是Git shell，github，Hexo，Markdown等等。虽然真的也只是接触而已。什么都不懂，只是沿着攻略，一步一步去做而已。我觉得未来会越来越了解的呀。开心嘻嘻嘻。

最后，激发了我对于前端的兴趣。自己知道的很浅，但是我已经感觉到那种好棒的感觉。自己这个专业因为太杂，总觉的自己的路子很多，可以去走技术方面，又可以走管理的路。可以思考去考研，也可以找工作。自己都有想法，摇摇摆摆像是家中老旧的风扇机头旋钮坏掉一摇一摆的样子一样……发现自己真的真的很能沉浸在计算机的世界。里面的东西让我着迷。很想就这样~想就要去做对吧跌娣。

Markdown学习小记

Markdown一种标记语言。之前在CSDN中接触过，但是就是单纯地觉得很好看，并没有觉得有什么，没想到这也是一种很棒的标记语言。突然有种时机未到的感觉啊。
下面讲下自己今天学到的东西：

  • Haroopad来编辑文章
  • “—”之内的都是显示在文章上面的东西，需要有，可以小小修改
  • 每一段前不用空两格，不然就是另一种形式啦，不会自动换行了，直接下一行开头。
    没啦，用的越多心得会越多的啦。期待，开心嘻嘻嘻。
